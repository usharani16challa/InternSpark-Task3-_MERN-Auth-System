Project Title: MERN Auth System (InternSpark Task)
-------------------------------------------------------------------
Tech Stack: React, Node.js, Express, MongoDB Atlas, JWT, Bcrypt.
-------------------------------------------------------------------
Features:
Secure User Registration & Login.
Password Hashing with Bcrypt.
Protected Frontend Routes using React Context.
Secure Token storage in httpOnly Cookies.
---------------------------------------------------------------------
How to Run: (Mention npm install and setting up the .env file).
